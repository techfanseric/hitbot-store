HitbotOS Store · 设计规范

统一 token + 组件库，确保官网商城、OS 端嵌入、管理后台视觉一致。

核心色 #BD1C22 主字体 Segoe UI（营销用庞门正道） 基础字号 12px 圆角 4-8px 边框 基本不用
