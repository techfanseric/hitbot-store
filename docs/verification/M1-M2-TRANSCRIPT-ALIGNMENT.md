## Remaining Gaps

- Real backend account/session API is still missing; the member-account entry is local and uses browser storage.
- Real cross-device cart/order sync is still missing; the OS BOM import and handoff inbox are currently local simulations.
- Real payment channel, invoice validation, logistics carrier API, and post-payment statuses are still local simulations; the current payment receipt is a deterministic local reference for workflow verification.
- OS-side cart generation still needs integration with the OS repository/backend contract.
- Real admin backend, product publishing APIs, server-side permission enforcement, and audit logs belong to the separate backend/admin workstream.
- Trial account anti-abuse, SMS verification, and real trial-account provisioning are still missing.
